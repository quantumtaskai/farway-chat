Quantum Chat Widget

AI-powered voice chat widget for your website

Just one line of code to add intelligent conversation to any website

Features

Voice Input

Natural speech-to-text using Web Speech API. Speak naturally and get instant transcription.

AI Responses

Powered by OpenAI GPT for intelligent, contextual responses to user queries.

Text-to-Speech

AI responses are automatically spoken aloud using natural voice synthesis.

Easy Integration

1. Basic Integration

Add this single line to your website's HTML:

<script src="https://quantumtaskai.github.io/quantumchat/embed.js"></script>

2. With Configuration

Customize position and behavior:

<script> window.quantumChatConfig = { position: 'bottom-left', // bottom-right, bottom-left, top-right, top-left autoOpen: false, // Auto-open widget on page load apiKey: 'your-api-key' // Pre-configure API key }; </script> <script src="https://quantumtaskai.github.io/quantumchat/embed.js"></script>

3. Programmatic Control

Control the widget with JavaScript:

<script> // Open the chat widget QuantumChat.open(); // Send a message programmatically QuantumChat.sendMessage('Hello from my website!'); // Set API key QuantumChat.setApiKey('your-openai-api-key'); // Close the widget QuantumChat.close(); </script>

4. Event Callbacks

Listen to widget events:

<script> window.quantumChatCallbacks = { onOpen: function() { console.log('Chat opened'); // Track analytics event }, onClose: function() { console.log('Chat closed'); }, onMessageSent: function(message) { console.log('User sent:', message); }, onMessageReceived: function(message) { console.log('AI replied:', message); } }; </script>
